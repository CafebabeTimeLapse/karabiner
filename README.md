# karabiner
My Karabiner-Element settings

## Requirements

- Mac OS
- Karabiner Element
  - Make sure that `Default profile` profile found in your Karabiner Element UI.

## How to install

`Default profile` is overwritten by

```
$ yarn apply:default
```

If you want to overwrite `'Your profile'`, run

```
$ yarn apply 'Your profile'
```
